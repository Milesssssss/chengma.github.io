CSS 单位

http://web.jobbole.com/82490/

rem

相对于根元素大小

em

相对于夫元素字体大小

vh 和 vw

  • vh : 等于 viewport 高度的 1/100
  • vw : 等于viewport 高度的 1/100

vmin 和 vmax

vmin和vmax和viewport 最大高度和宽度有关

ex 和 ch

与em和rem相似， 依赖于当前字体和字体大小。

不同的是 两个单位只依赖于font-family，基于特定字体的法案。

ex单位被定义为”当前字体的x-height或者一个em的一半”
