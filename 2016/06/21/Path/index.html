Path

path.join()

用于连接路径。
会正确使用当前系统的路径分隔符。
unix “/“ windows “\”

path.resolve()

相对路径转为绝对路径。
除了根目录，返回值都不带斜杠

accessSync()

同步读取一个路径

判断一个目录是否存在

path.relative

接受两个参数，都是绝对路径，返回第二个路径相对于第一个路径的那个相对路径。

在第一个目录进入第二个目录。
