git 命令笔记

基本命令

  • git clone
  • git add
  • git status
  • git commit
  • git remote
  • git fetch
  • git pull
  • git push

$ git clone

$ git clone <仓库网址> <本地目录名>

$ git add

git add <file>

$ git status

查看文件修改状态

$ git commit

$ git commit –amend

上面操作只会产生一次提交
第二次 add 的文件会添加到commit中。

取消已经暂存的文件

不小心git add了不想提交的内容

撤销修改

如果觉得一个文件没有必要修改 想回复到之前版本。

删除文件

$ git rm test.txt

恢复：

$ git checkout -- test.txt

远程仓库 git remote

$ git remote add origin git@github.com:michaelliao/learngit.git

添加后 远程仓库的名字就是 origin 也可以改成别的东西

远程仓库回滚

假设你有3个commit如下：

其中最后一次提交commit 3是错误的，那么可以执行：

git reset --hard HEAD~1

会发现，HEAD is now at commit 2。

然后再使用git push –force将本次变更强行推送至服务器。这样在服务器上的最后一次错误提交也彻底消失了。
